## معرفی

**V2Ray Free Configs** یک پروژهٔ خودکار است که کانفیگ‌های معتبر **VMess**, **VLess** و **Trojan** را از منابع تلگرامی جمع‌آوری کرده و به‌صورت مرتب به‌روزرسانی می‌کند.

این ریپازیتوری برای ارائهٔ کانفیگ‌های رایگان و قابل استفاده برای کاربران ایرانی ایجاد شده است.

https://raw.githubusercontent.com/awnirosen2/awmir_v2ray/refs/heads/main/configs.txt

۳. لینک را اضافه کنید و روی Update بزنید.

**به‌روزرسانی:** این کانفیگ‌ها هر **۴ ساعت** یک‌بار به‌روزرسانی می‌شوند.
